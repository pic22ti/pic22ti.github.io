## 4. 사용된 기술
* CSS
  - 미디어쿼리를 사용한 반응형 사이트 구현
  - grid 레이아웃을 사용한 디자인 구현
  - flex 레이아웃 

* Vanilla JS (단어 고치기)
  - url을 사용하여 선택한 메뉴로 이동
  - scroll 위치를 구해 gototop 버튼 활성화
  - 선택된 탭을 보여주기
  - 게시글 슬라이드 다운
  - 모달 팝업창 구현

## 5. 고찰 및 개선 방향
* 컨텐츠를 전부 html로 직접 넣어 유지보수에 효율적이지 않음
